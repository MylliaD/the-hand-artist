# Blueprint: The Hand Artist

> Generated by The Architect

## 1. Project Overview
**The Hand Artist** is a high-performance web application that leverages **MediaPipe Hands** to allow users to draw and interact with 2D objects in real-time using only their hands and a webcam.

## 2. Tech Stack
- **Framework:** Vite + React 19 (TS)
- **Vision Engine:** MediaPipe Hands v0.10 (Wasm)
- **Canvas:** Layered Canvas (UI Layer, Drawing Layer, Interaction Layer)
- **State:** Zustand (Coordenadas de landmarks y modo de pincel)
- **Styling:** Tailwind CSS

## 3. Interaction Logic
### Pinch (Grab/Move)
- **Condition:** Euclidean distance between Index Tip (Landmark 8) and Thumb Tip (Landmark 4) < 0.05.
- **Action:** Selects object under coordinates or continues moving selected object.

### Index Point (Draw)
- **Condition:** Index Tip (8) is raised (Y8 < Y6) AND Thumb Tip (4) is far from Index Tip.
- **Action:** Records points in the current path.

## 4. Build Order
1.  **Setup Canvas Component:** Layered approach with `requestAnimationFrame`.
2.  **MediaPipe Integration Hook:** `useHands` hook to manage the webcam stream and Wasm model loading.
3.  **Drawing Engine:** Implement line smoothing (Catmull-Rom or simple averaging) to avoid jitter.
4.  **Interaction System:** Basic rectangle objects that can be moved with a Pinch gesture.

# Instructions for Gemini (Builder)

You are building 'The Hand Artist'. Follow these rules:

1. **Performance First:** Use `requestAnimationFrame` for all canvas updates. Avoid using React State for coordinates; use a mutable object or Zustand with external store updates.
2. **MediaPipe Setup:** Use the CDN version of MediaPipe assets for faster initial load.
3. **Responsive Canvas:** Ensure the canvas resolution matches the video aspect ratio (usually 4:3 or 16:9).
4. **Drawing Logic:** When the index finger is raised, draw a continuous line. Clear the current path when the finger is lowered.
